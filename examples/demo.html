Using a grit wrapper

A grit wrapper can be used to make your layout more readable by limiting the maximum width it can occupy. This class should only be used one time and placed around your top level grit element!
In this example we use the grit-wrapper-md to limit the grit width to the default tablet breakpoint width (768px).
Possible wrapper sizes include:

  • .grit-wrapper-sm max. 320px
  • .grit-wrapper-md max. 768px
  • .grit-wrapper max. 1200px (no suffix because this is the opinionated default)
  • .grit-wrapper-xl max. 1680px

Adding & positioning content

Adding content into a grid can be done in several ways depending on the type of layout you want to achieve. For any layout though, you can add as many direct children elements to the grit element as you want. These will be positioned beneath each other by default.
The following sections explain each type of class which can be added to a direct child of a grit or subgrit element.

.subgrit

The subgrit class will convert its element into a grit itself, inheriting setting such as column count and padding, filling the entire parent grit ignoring its padding (Padding is ignored to ensure the grids are the exact same size).
More information on the subgrit class can be found inside the subgrid section.

.content

The content class will make the element fill the whole available width, whilst enforcing the opinionated default padding (mobile: 1 column | >= tablet: 2 columns).

.grt-fill

The grt-fill class will make the element fill the whole available width, whilst enforcing the padding used by the grit.

.grt-full

The grt-full class will make the element fill the whole available width, ignoring any padding.

.grt-a-b

The grt-a-b (a: number 1-32, b: number: 1-32) class will make the element span across the columns defined by a and b.

.overlay

The overlay class will mark the element as an overlay. This can be useful if you have multiple direct children of a grid and you want to position these children on top of each other. After adding this class, the elements will be stacked.
Note: It can be useful to combine this class with the .subgrit class in order to create stacked layouts inside the grit.
